#2. ЦЕЛЬ  	
Разработать ai ассистента который в цикле проверяет файл task.txt, вытаскивает данные связанные с заданием, шаблон задания прилагается. Далее проверяет если задача не выполненна то запускает процесс работы с заданием.
После получения задания, ассистент отправляет запрос в lm studio запрос на решение данной задачи, после получения ответа от llm мы делаем те операции которые сообщила нам llm.
Ассистент должен делать задачи в цикле.
Ассистент должен пройти несколько этапов при решении задачи:
1) Получение данных с task.txt.
2) Отправка запроса по задаче в llm.
3) Получение и обработка ответа от llm.
4) Реализация действий которые прислала llm.
5) Компиляция полученного кода.
6) Написание тестов при необходимости.
7) Запись действий и причины написания именно такого кода в файл log_task_*.txt.
8) Если прошли все тесты по данной задачи то переходим к следующей задачи. Если не прошли тесты то выполняем данную задачу заново но добавляем в запрос к llm информацию по коду и действиям которые ранее были сделаны.

- 1) Создание файлов, внесение изменений в файл, чтение файлов, удаление файлов.
