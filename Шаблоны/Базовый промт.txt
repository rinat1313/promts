# ----- 1. КОНТЕКСТ  -----
Я программист, хочу с помощью AI написать ассистента который может по циклу обрабатывать файл task.txt и по очередно вытаскивать данные по свободной задаче
и добавлять к себе в обработку. Язык программирования используем Go.

# ----- 2. ЦЕЛЬ  -----
Мне нужно что бы ассистент имел возможность вытаскивать данные по свободной задаче с task.txt и анализировал входящие параметры:
1) Сама задача, в ней подробно описано что ожидается от ассистента;
2) Ожидаемый результат, в ней подробно описано что ожидается при выполнении написаного кода;
3) Примеры тестов, в нем подробно описано что будет как входящие параметры тестов и какие параметры ожидаются;
4) Важные моменты, в нем указываются моменты которые надо точно учитывать при написании и выполнении задачи.

# ----- 3. ДЕТАЛИ  -----
При выполнении задачи обязательно учти следующие нюансы:
Надо подробно проанализировать Задачу;
Обязательно надо описать в конце почему ты написал такой код.

# ----- 4. ФОРМАТ ВЫВОДА  -----
Ответ оформи строго по этим требованиям:
1) Сначала идёт код в Go.
2) В конце идёт txt файл где описано почему написан именно такой код.
3) Объём без ограничений.

# ----- 5. ПРИМЕРЫ (FEW-SHOT)  -----
Мне нужен ответ, похожий на этот пример:
Вход: Напиши код на Go который должен вернуть текст "Привет мир!".
Выход: 

package main

    fmt.Println("Привет мир!")
}
Сделай так же с моими данными.
